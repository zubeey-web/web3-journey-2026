# Day 1: What is Ethereum?

**Ethereum** = A global computer that no one owns. It runs apps called smart contracts that can’t be shut down.

**Node** = One computer out of 10,000+ that stores all Ethereum data and checks every transaction. Nodes = security.

**Gas** = Fee paid in ETH to use the network. Higher gas = faster transaction. Stops spam.

**Why Web3 frontend devs need this**: 
As a React developer, I build UIs that connect to MetaMask. If a user’s transaction fails, I need to explain gas or node errors. This knowledge lets me debug Web3 apps and build trust with users.
